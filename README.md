# Pharmaceutical Company Web Application

This is a modern web application template for a pharmaceutical company, built with HTML, CSS, and TypeScript.

## Features
- Professional, clean UI
- Modular structure for scalability
- Ready for business customization

## Getting Started
1. Open the project in VS Code.
2. Run a local server (e.g., with Live Server extension or `npx serve`).
3. Edit the HTML, CSS, and TypeScript files in the `src` folder.

## Project Structure
- `src/` - Source files (HTML, CSS, TS)
- `dist/` - Compiled output (if using a bundler)
- `.github/` - Project instructions

## Customization
Replace placeholder content and styles with your company's branding and requirements.
